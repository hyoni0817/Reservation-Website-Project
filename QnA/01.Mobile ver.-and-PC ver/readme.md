## 사용법
1. 01.Mobile ver.-and-PC ver 경로에서 app.js를 실행시켜줍니다.
   ```
   node app.js
   ```
2. app.js를 실행하고 난 뒤에 아래 URL을 통해서 글 목록과 작성글을 볼 수 있습니다.   
**🖥URL**   
글 목록보기 : <http://localhost:3000/>   
글 보기 : <http://localhost:3000/qnaView>
